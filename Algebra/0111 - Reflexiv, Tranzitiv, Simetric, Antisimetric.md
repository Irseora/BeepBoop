- **(R) Reflexiva**: $a \ \rho \ a, \hspace{3mm} \forall a \in A$
- **(T) Tranzitiva**: $a \ \rho \ b \ \text{ si } \ b \ \rho \ c \Rightarrow a \ \rho \ c$
- **(S) Simetrica**: $a \ \rho \ b \Rightarrow b \ \rho \ a$
- **(A) Antisimetrica**: $a \ \rho \ b \ \text{ si } \ b \ \rho \ a \Rightarrow a = b$
